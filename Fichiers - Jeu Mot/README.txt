Compilation : "gcc -Wall main.c -o main -lm" cette commande permet de créer selon le terminal un fichier executable main associe au programme main.c

Execution : "./main" afin d'executer le fichier executable du programme main.c

Doxygen : "doxygen -g" permet de creer un fichier editable pour la personnalisation du fichier final, 
puis "doxygen Doxyfile" permet de creer le fichier html et Latex du programme main.c
Dans le html -> index.html pour voir le resultat ou convertir refmain.tex en PDF pour obtenir une autre forme d'affichage du resultat.
