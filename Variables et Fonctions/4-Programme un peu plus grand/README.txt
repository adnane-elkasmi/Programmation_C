Compilation : "gcc -Wall Algorithme4.c -o Algorithme4" cette commande permet de créer selon le terminal un fichier executable Algorithme4 associe au programme Algorithme4.c

Execution : "./Algorithme4" afin d'executer le fichier executable du programme Algorithme4.c

Doxygen : "doxygen -g" permet de creer un fichier editable pour la personnalisation du fichier final, puis "doxygen Doxyfile" permet de creer le fichier html et Latex du programme Algorithme4.c
Dans le html -> index.html pour voir le resultat ou convertir refmain.tex en PDF pour obtenir une autre forme d'affichage du resultat.
