Compilation : "gcc -Wall Algorithme7.c -o Algorithme7 -lm" cette commande permet de créer selon le terminal un fichier executable TP1 associe au programme Algorithme7.c

Execution : "./Algorithme7" afin d'executer le fichier executable du programme Algorithme7.c

Doxygen : "doxygen -g" permet de creer un fichier editable pour la personnalisation du fichier final, puis "doxygen Doxyfile" permet de creer le fichier html et Latex du programme Algorithme7.c
Dans le html -> index.html pour voir le resultat ou convertir refmain.tex en PDF pour obtenir une autre forme d'affichage du resultat.

NB: Sous linux, avec gcc cela revient à ajouter l'option -lm pour utiliser la librairie math !
