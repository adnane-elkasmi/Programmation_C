Execution : "./Algorithme6.1" afin d'executer le fichier executable du programme Algorithme6.1.c

Doxygen : "doxygen -g" permet de creer un fichier editable pour la personnalisation du fichier final, puis "doxygen Doxyfile" permet de creer le fichier html et Latex du programme Algorithme6.1.c
