Compilation : "gcc -Wall Algorithme1.c -o Algorithme1" cette commande permet de créer selon le terminal un fichier executable Algorithme1 associe au programme Algorithme1.c

Execution : "./Algorithme1" afin d'executer le fichier executable du programme Algorithme1.c

Doxygen : "doxygen -g" permet de creer un fichier editable pour la personnalisation du fichier final, puis "doxygen Doxyfile" permet de creer le fichier html et Latex du programme Algorithme1.c
Dans le html -> index.html pour voir le resultat ou convertir refmain.tex en PDF pour obtenir une autre forme d'affichage du resultat.
